## üìñ What is This?

Sailarr Installer provides a fully automated installation for creating your own media server. Using this software, you can stream content powered by Real-Debrid through Plex, while managing everything with the *Arr applications such as Radarr, Sonarr, and Prowlarr. The best part is that it does all the work for you based on your choices.

## üöÄ Key Features

- **Fully Interactive Setup:** Customize your setup with exactly what you need.
- **Zero Manual Configuration:** Enjoy hassle-free installation as all services are automatically configured.
- **TRaSH Guide Quality Profiles:** Access industry-standard quality settings via Recyclarr.
- **Health Monitoring:** Rest easy knowing that the installer automatically restarts containers if any issues arise.
- **Decypharr Integration:** Use a lightweight Real-Debrid download client with symlink support.
- **Optional Traefik Integration:** Implement a reverse proxy with HTTPS support for added security.
- **Authentication:** Enhance security with optional password protection for your services.
